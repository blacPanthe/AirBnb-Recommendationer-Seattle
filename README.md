# Airbnb Listing Recommendation System - Seattle

NLP-powered recommendation system built on 541K guest reviews from the Seattle Airbnb dataset. Uses BERTopic for semantic topic modeling to group listings by stay experience, combined with VADER sentiment analysis. Includes a Flask web app with an interactive topic browser.

## How It Works

1. **541K reviews** are cleaned, POS-tagged (proper nouns removed), and aggregated per listing
2. **BERTopic** (sentence-transformers + UMAP + HDBSCAN) discovers semantic topics from review text
3. **VADER** sentiment scores are computed per review and averaged per listing
4. **Flask app** serves a topic browser UI — click a topic to see top-rated listings in that category

## Setup

### 1. Download the data
Download these two files from [Inside Airbnb — Seattle](http://insideairbnb.com/get-the-data/) and place them in the project root:
- `listings.csv`
- `reviews.csv`

### 2. Install dependencies
```bash
pip3 install flask bertopic sentence-transformers umap-learn hdbscan nltk scikit-learn numpy pandas certifi
```

### 3. Train the models (one time, ~10 min)
```bash
python3 preprocess.py
```

### 4. Run the app
```bash
python3 app.py
```
Open **http://localhost:8080** in your browser.

## Tech Stack

- **BERTopic** — topic modeling (UMAP + HDBSCAN + sentence-transformers)
- **NLTK / VADER** — text preprocessing and sentiment analysis
- **scikit-learn** — cosine similarity, TF-IDF
- **Flask** — web server
- **Vanilla JS** — frontend, no build step
